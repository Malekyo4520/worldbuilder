## 📦 What is worldbuilder?

worldbuilder is a desktop app for building fictional worlds from your computer. It gives you a command line tool, a web interface, and Claude Code integration in one place.

Use it to create:

- novels
- series bibles
- D&D campaigns
- game worlds
- deep lore and setting notes
- character sheets
- location lists
- timelines

It keeps your world data in files, so you can move it, back it up, and edit it with other tools. It also supports AI help for structure, writing support, local image generation, and character voices.

## 🔐 File safety and backups

Because your project lives in regular files, you should keep a backup copy. A simple backup plan is:

- copy the whole project folder to another drive
- keep one backup on an external disk
- save a second copy in cloud storage if you use it

This helps protect your work if your computer has a problem.

## 🧩 Working with large worlds

If your project gets big, split it into clear parts:

- one file for each major character
- one file for each city or region
- one file for each faction
- one file for each timeline era

That keeps the project easier to read and update. It also helps the app stay organized as the world grows.

## 🎨 Using images and voices

If you want pictures or voice assets, create them after your core world is in place.

A good order is:

1. define the world
2. define the people in it
3. define the important places
4. then create art and voices

This gives you better results and keeps your setting consistent.
